v1.0.0 - Initial Release

Azokle Auth is a secure and privacy-focused authentication app designed to help users protect their tokens and accounts. The app supports multiple authentication methods, including password protection, biometrics, and secure backups, ensuring a seamless and safe user experience.

  • Setup Azokle Auth with an easy-to-follow guide
  • Biometric authentication (fingerprint/face recognition)
  • Password protection for vault security
  • Manual and cloud backup options (for encrypted vaults)
  • Support for importing from FreeOTP+
  • Auto-lock feature for enhanced security
  • Export and import vault data across devices
